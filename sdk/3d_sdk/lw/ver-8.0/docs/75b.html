7.5b Changes

February 18, 2003

This is a list of the changes in the LightWave 7.5b release that affect the SDK.

lwlcore.h

  • This is a new header file containing the definitions to allow native-code plug-ins to acquire and use the new LScript Universal Requester system.

lwlscript.h

  • This is a new version of an old header usually distributed separately. It is used to develop native-code function and User-defined Object Agent libraries for LScript.

Miscellaneous

  • A new project, lsur, has been added to the SDK samples that demonstrates the usage of the LScript Universal Requester mechanism.
